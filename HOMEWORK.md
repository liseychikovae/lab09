# Домашнее задание lab09

Лабораторная работа Лисейчиковой Екатерины

## Что требовалось сделать

Домашнее задание относится к теме лабораторной работы IX: изучение сервисов хранения артефактов на примере github release.

## Выполненные пункты

- артефакты создаются из CPack-конфигурации, подготовленной в lab06;
- релизный workflow не хранит токены в репозитории, используется стандартный `GITHUB_TOKEN`; 
- имя тега становится точкой входа для публикации версии;
- скрипт можно запускать локально в Linux-окружении до создания тега.

## Как проверить домашнее задание

```bash
bash scripts/package_release.sh
git tag v0.1.0
git push origin v0.1.0
ls _build/*.tar.gz
```

## Пояснение по реализации

Проект оформлен так, чтобы его можно было перенести в Linux-окружение и собрать стандартными командами CMake. Файлы не завязаны на абсолютные пути macOS, персональные секреты не добавлены в git, а дополнительные инструменты подключаются через конфигурационные файлы конкретной лабораторной.
